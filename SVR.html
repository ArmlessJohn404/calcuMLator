Support Vector Regression

The Support Vector Regression utilizes the concept of support vectors. It adds another layer of complexity on top of the Linear Regression, which makes it possible to create very complicated curves to fit the data. In our case, it approximated the funcions in the range of the training set with a high variance. Outside the range of the training set the algorithm predicts 0.
